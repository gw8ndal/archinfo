<h3 align=center>A simple yet beautiful fetch script made for Arch Linux<h3>

## Installation

- Clone this repo

```git clone https://github.com/gw8ndal/archinfo && cd archinfo```

- Execute the install script

```bash install.sh```

## Uninstalling

- Run the uninstall script

```bash remove.sh```
